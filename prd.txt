# 《知卡研习》AI智能学习产品 PRD产品需求文档
**文档版本**：V1.0
**产品定位**：基于AI Agent+知识卡片体系，颠覆传统长线学习，实现「核心概念深挖+内化输出+自查纠偏+体系速成+长效复习」全链路高效学习工具，覆盖学生应试、职场转行、行业深耕、认知提升全人群
**开发模式**：内置专属Skill技能组，本地素材库自定义路径接入，AI全流程协同交互

---

## 一、产品核心目标
1. 砍掉传统学习80%无效冗余内容，直击**核心关键词/考点/行业底层概念**
2. 统一学习逻辑：深挖概念→生成知识卡→人机研讨融合→口述输出→AI对标纠错打分
3. 实现极速学习：3-5天吃透一学期学科主干，7天搭建完整行业认知体系
4. 搭建个人专属知识资产库，自动归档学习轨迹，搭配科学复习永不遗忘
5. 适配AI时代趋势，培养「决策判断+边界界定+逻辑梳理」AI不可替代核心能力

## 二、目标用户分层
| 用户群体 | 核心需求 | 适配学习模式 |
| ---- | ---- | ---- |
| 中小学生 | 课内考点速成、考前冲刺、弱科补齐、快速提分 | 全科考点词库+答题模板+错题复盘 |
| 初高中升学党 | 精简备考、避开无用知识点、构建学科体系 | 重难点分级学习+应试出题视角学习 |
| 职场新人/转行人群 | 快速入行、吃透行业逻辑、面试通关 | 行业核心词库+实战话术+场景案例 |
| 职场资深人士 | 跨界拓认知、高阶思维提升、打破职业瓶颈 | 高阶概念研习+跨领域知识联动 |
| 认知成长人群 | 心理学/经济学/商业思维深度精进 | 自由文案研习+深度思辨研讨 |

## 三、产品整体架构（五大核心功能模块）
### 模块1：素材资源中心（底层数据源）
1. **素材分类库**
   - 校内学科库：语数英物化政史地生全学段核心考点词库
   - 行业赛道库：市场营销、金融、心理学、运营、电商等，词汇量灵活设定80-300个/行业
   - 自由研习库：支持导入抖音逐字稿、公众号干货、课程文案、书籍节选
2. **素材接入规则**
   - 支持用户**自定义本地文件夹/文档路径**，自行上传整理学习素材
   - 平台内置公共免费词库，用户可自建私人专属小众领域词库
3. **词汇难度分级**
   入门级（基础必学）、进阶级（核心提能）、高阶级（拔高竞赛/高薪进阶）

### 模块2：核心研习主流程（产品核心Skill技能）
#### 流程1：选定学习内容
1. 用户选择学习专区：全科学习/行业速成/自由研习
2. Agent调取对应素材列表，用户挑选今日学习概念/关键词
3. 确定当日学习数量（自定义5-30个）

#### 流程2：生成初代标准知识卡片
1. AI拆解原文素材，提炼正统核心内容
2. 固定知识卡片统一模板
   - 基础信息：主题、来源、所属领域
   - 核心定义：精准官方释义
   - 适用边界：能用场景+禁用场景
   - 原始观点：素材原文核心思想
   - 基础案例：通用基础用法
   - 常见误区：大众普遍错误理解

#### 流程3：深度人机深究研讨
1. 用户自由提问、抒发个人见解、延伸思考
2. AI配合双向拉扯交流：补充底层逻辑、新增实战案例、细化边界、区分易混概念
3. 破除用户原有错误认知，完成思维融合

#### 流程4：生成用户专属融合知识卡
整合原文内容+研讨内容+个人理解+新增场景，生成**私有化内化新版卡片**，区别通用标准版

#### 流程5：自主输出+AI智能对标审核
1. 用户静心思考后，以**文字/语音口述**形式完整讲解概念
2. AI对比「用户输出内容+专属融合知识卡」完成全维度核查
3. 输出审核结果：
   - 四维量化打分（满分100）：定义准确度、边界清晰度、案例完整度、误区认知度
   - 逐条标注：思维漏洞、理解偏差、知识点遗漏、逻辑断层
   - 给出修正话术与标准正确解读

### 模块3：历史学习档案馆
1. 全自动归档所有学习记录，时间线可视化展示
2. 归档留存内容：学习时间、所选概念、原文素材、初代卡片、融合卡片、个人口述内容、AI纠错评分报告
3. 支持检索、筛选、一键回看整套学习流程
4. 统计学习数据：已学词汇量、掌握平均分、薄弱知识领域

### 模块4：智能复习体系
1. **艾宾浩斯定时复习**：1天/3天/7天/15天自动推送已学内容
2. 双复习模式
   - 速览模式：快速浏览知识卡片轻量化复盘
   - 闭卷口述模式：二次口述作答，重新打分巩固记忆
3. **专属思维错题本**
   单独收纳所有学习中出现的理解偏差、逻辑漏洞、知识盲区，优先定向复盘，根治顽固错误认知
4. 阶段性综合结业测评：完成整套词库学习后，统一摸底检测整体掌握程度

### 模块5：实战落地&输出变现模块
1. **场景绑定填充**：所有知识卡强制匹配职场/应试真实应用场景
2. 学生端：配套对应练习题、标准化答题模板、考试高频考点标注
3. 职场端：自动生成行业沟通话术、面试应答文案、工作实操思路
4. 一键输出：讲课逐字稿、面试演讲稿、学习总结文案、自媒体创作文案

## 四、细分专区详细功能
### 1）中小学全科学习专区
1. 覆盖小学/初中/高中全九大学科
2. 理科专属卡片优化：公式+适用条件+易错计算点+解题思路
3. 文科专属卡片优化：答题框架+得分术语+素材运用+默写要点
4. 逆向出题视角：标注知识点出题频率、分值占比、必考题型

### 2）行业极速速成专区
1. 单行业精简核心关键词，拒绝冗余知识
2. 学习节奏：单日15-20词，一周完成全行业体系搭建
3. 自动串联行业词汇逻辑关系，生成行业全局思维导图
4. 补充行业潜规则、职场实操经验、入行必备实操技巧

### 3）自由深度研习专区
1. 无固定词库限制，用户任意导入干货文案、口播逐字稿
2. 沿用统一研习流程，自定义深挖任意认知概念
3. 支持跨行业、跨学科知识联动打通

## 五、产品核心优势&解决行业痛点
### 优势
1. 学习效率提升5-10倍，彻底摆脱逐字啃书低效模式
2. 人人统一标准化学习流程，零基础快速上手
3. AI实时即时纠错，反馈速度远超线下老师
4. 知识永久沉淀为个人资产，可复用、可复盘、可输出
5. 全人群通吃，学生应试、职场转行、自我提升全覆盖

### 解决痛点
1. 传统学习周期长、内容杂、抓不住重点
2. 只会死记概念，不懂边界、不会落地、实战脱节
3. 学习内容零散，无法搭建完整知识体系
4. 错题、思维误区无法集中整理，反复犯错
5. AI时代普通人只会执行，缺少顶层判断思维

## 六、风险规避&优化预案
1. **重骨架轻细节缺陷**
   预案：先速成搭建骨架，增设「细节补充专区」，按需精细化深挖溯源知识
2. **低龄儿童适配性弱**
   预案：低龄版简化逻辑思辨，改为趣味短句卡片，侧重基础记忆
3. **自律性差用户拖延**
   预案：增设每日学习打卡、学习定额、惰性提醒、进度倒逼机制
4. **纯理论脱离实操**
   预案：每一个概念绑定实战任务，学完立刻落地练习，杜绝纸上谈兵

## 七、AI时代长期产品规划（5-10年适配布局）
1. 短期：完善全学科+全行业基础词库，打通全流程研习交互
2. 中期：强化跨领域知识联动引擎，实现一理通百业
3. 长期：主打「驾驭AI学习」，让用户借助本产品成为AI流程搭建者、规则定义者，远离被AI替代的纯执行类工作
4. 贴合未来教育改革趋势，轻量化、核心化、理解式学习成为主流，产品提前占领市场赛道

## 八、版本迭代规划
1. V1.0 正式版：完成核心研习流程、素材库、历史归档、基础复习功能
2. V1.1 优化版：新增四维打分体系、思维错题本、行业思维导图自动生成
3. V1.2 全量版：补齐中小学全学段词库、全主流行业词库、实战答题/职场话术库
4. V2.0 进阶版：上线高阶思维研习、知识跨界联动、私人自定义知识库全套功能

## 九、二期功能清单（Phase 2）

### 功能：导航重构 + 社群大厅

**背景**：当前学习中心是万花筒（内置词库+自定义资料+社群内容混在一起），交互不够清晰。

**目标**：
- 左侧栏拆分为「学习中心 / 复习 / 社群大厅」三个一级菜单
- **学习中心**：我正在学的（个人词库）—— 专注当前学习任务
- **复习**：归档的学习记录 + 艾宾浩斯待复习内容 —— 复习管理
- **社群大厅**：所有用户共享的资料知识库 —— 拉通各用户资料

**核心价值**：
- 个人学习路径清晰，不被海量社群内容干扰
- 为后续用户资料互通、UGC词库共享打基础
- 长期可扩展为知识市场/学习社群产品形态
